
# Proyecto personal (portafolio)


Portafolio personal desarrollado con HTML, CSS y JavaScript puro. Diseño minimalista en blanco y negro, totalmente responsive para dispositivos móviles y desktop.

Nacio como hobby para la reorganizacion de mis proyectos, y se me ocurrio darle una forma más seria y profesional : ) 

¿Que ocupo?  
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (Vanilla)
- Font Awesome 7

Normalmente suelo trabajarlos los fines de semana, asi que el proceso de creacion final es algo tardo : ) y durante lo hago refuerzo y aprendo
